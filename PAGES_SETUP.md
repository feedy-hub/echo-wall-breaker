# GitHub Pages setup

This repository already contains a bilingual showcase page.

## Enable Pages

1. Open repository Settings.
2. Go to Pages.
3. Source: Deploy from a branch.
4. Branch: main.
5. Folder: /docs.
6. Save.

After GitHub finishes deployment, the page will be available at:

https://feedy-hub.github.io/echo-wall-breaker/
