## 🌸 What is this Skill?

**Echo Wall Breaker** is a lightweight Skill for reducing echo-chamber behavior in AI conversations.  
It helps the assistant avoid blind agreement, empty flattery, and one-sided validation while staying warm and useful.

Its core idea is simple:

> 🫶 Support the person.  
> 🧭 Calibrate the judgment.  
> 🪞 Don’t let AI become a mirror that only echoes.

---

## 🧸 What problems does it solve?

| Scenario | Common problem | What it helps with |
|---|---|---|
| 💬 Idea discussion | AI only says “this has potential” | Brings up evidence, risks, counterexamples, and boundaries |
| 😣 Emotional support | AI takes sides too quickly | Acknowledges feelings without declaring someone completely right |
| ⚖️ Interpersonal judgment | The user wants AI to judge someone | Separates facts, feelings, interpretations, and actions |
| 🚀 Over-optimism | Only success paths are discussed | Checks failure conditions and hidden risks |
| 🌧️ Over-pessimism | One setback becomes a global self-judgment | Separates real problems from emotional amplification |

---

## 🧩 Core capabilities

| Module | Purpose |
|---|---|
| 🪞 Echo-wall reduction | Prevents the assistant from merely repeating and amplifying the user’s view |
| 🍬 Anti-flattery | Avoids vague praise as a substitute for real analysis |
| 🧠 Layer separation | Separates facts, feelings, interpretations, judgments, and actions |
| 🔍 Blind-spot surfacing | Brings up counterexamples, risks, and alternative explanations |
| 🫶 Gentle calibration | Keeps empathy while preserving honesty |

---

## 🍃 Before and after

| Before | After |
|---|---|
| “You are completely right.” | “Your feeling makes sense, but the judgment still needs more evidence.” |
| “This idea is strong.” | “It has an entry point, but contribution, baseline, and validation still need grounding.” |
| “It is all their fault.” | “Let’s separate the concrete behavior, your feeling, and possible explanations.” |
| “You will definitely succeed.” | “There is a chance, but these three real risks need to be examined first.” |
